## What it is

`octofhir-sof` is a Rust toolkit for
[SQL on FHIR v2](https://build.fhir.org/ig/FHIR/sql-on-fhir-v2/): it parses
`ViewDefinition` resources and turns FHIR data into flat tables. The library is
network-free and embeddable; the CLI is a thin shell over it.

- **Database-free in-memory evaluator** over `serde_json`, and a **PostgreSQL /
  DuckDB SQL generator** — both pass the full official content-test suite
  (144/144).
- **Version-agnostic execution**: the engine navigates resource JSON directly,
  so the same view runs unchanged over R4, R4B, R5 or R6.
- **Writers**: CSV, NDJSON, JSON, and Parquet (feature-gated).
- **Full v2.1 support**: the `repeat` directive and the `%rowIndex` environment
  variable, on both paths.

## Install

```sh
curl -fsSL https://raw.githubusercontent.com/octofhir/sof/main/install.sh | sh
```

See [Install](/sof/install/) for `cargo install`, prebuilt binaries, and
building from source.
